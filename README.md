# Portfolio — Marcos Queiroz Pereira

Portfólio profissional desenvolvido para apresentar minha trajetória, experiências, certificações, competências técnicas e conteúdos sobre **Salesforce Marketing Cloud, automação, SQL, Data Cloud e desenvolvimento web**.

## 🔗 Acesse o projeto

[Ver portfólio online](https://marcosqueirozpereira.github.io/portfolio/)

## ✨ O que você encontra no site

- Apresentação profissional
- Resumo da minha experiência
- Stack técnica e competências
- Certificações
- Conteúdo e dicas na área de tecnologia
- Meios de contato

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## 🎯 Objetivo do projeto

Criar uma vitrine profissional clara, moderna e direta, reunindo:
- posicionamento profissional
- histórico técnico
- provas de conhecimento
- presença digital para recrutadores e clientes

## 📁 Estrutura do projeto

- `index.html` — página inicial
- `sobre.html` — resumo profissional e formação
- `skills.html` — competências e stack
- `exp.html` — experiência profissional
- `certs.html` — certificações
- `blog.html` — conteúdos e artigos
- `contato.html` — formas de contato
